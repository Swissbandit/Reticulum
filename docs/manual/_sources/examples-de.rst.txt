.. _examples-main:

*************
Code Beispiele
*************

Eine Reihe von Beispielen ist in der Quelldistribution von Reticulum enthalten.
Sie können diese Beispiele verwenden, um zu lernen, wie Sie Ihre eigenen Programme schreiben.

.. _example-minimal:

Minimal
=======

Das Beispiel *Minimal* demonstriert die Mindestvoraussetzungen für die Verbindung mit einem
Reticulum-Netzwerk aus Ihrem Programm heraus herzustellen. In etwa fünf Zeilen Code werden Sie
den Reticulum Network Stack initialisiert und bereit haben, den Datenverkehr in Ihrem
Programm weiterzuleiten.

.. literalinclude:: ../../Examples/Minimal.py

Dieses Beispiel ist auch unter `<https://github.com/markqvist/Reticulum/blob/master/Examples/Minimal.py>`_ zu finden.

.. _example-announce:

Ankündigung.
========

Das Beispiel *Ankündigung* baut auf dem vorherigen Beispiel auf und zeigt, wie man
ein Ziel im Netz ankündigt, und wie Ihr Programm Benachrichtigungen über
Ankündigungen von relevanten Zielen erhält.

.. literalinclude:: ../../Examples/Announce.py

Dieses Beispiel ist auch zu finden unter `<https://github.com/markqvist/Reticulum/blob/master/Examples/Announce.py>`_.

.. _example-filetransfer:

Filetransfer
============

Das Beispiel *Filetransfer* implementiert ein einfaches Dateiserverprogramm,
welches Clients ermöglicht, sich zu verbinden und Dateien herunterzuladen. Das Programm verwendet die Schnittstelle Resource, 
um Dateien beliebiger Größe effizient über einen Reticulum :ref:`Link<api-link>` zu übertragen.

.. literalinclude:: ../../Examples/Filetransfer.py

Dieses Beispiel ist auch zu finden unter`<https://github.com/markqvist/Reticulum/blob/master/Examples/Filetransfer.py>`_.
